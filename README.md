## India’s International Trade Outlook: A Tableau Dashboard on Import & Export Dynamics

This project transforms multidimensional macroeconomic datasets into an interactive Tableau dashboard, providing a comprehensive overview of India’s global trade footprint. By visualizing millions of trade records across varying timeframes, this project makes it easy to analyze India's trade balance, identify lucrative export markets, and track critical import dependencies.

**Key Features:**
* **Geospatial Trade Mapping:** Choropleth maps detailing India's top international trading partners and pinpointing domestic state-level economic contributions.
* **Commodity & Sector Analysis:** Interactive bar charts highlighting the top 10 highest-grossing export and import sectors (e.g., Petroleum, Electronics, Agriculture).
* **Trade Balance Tracking:** Grouped bar charts offering a direct comparison of Export vs. Import values to instantly identify regional and sector-specific trade deficits or surpluses.
* **Multi-Level Granularity:** Seamless drill-down capabilities from high-level annual macro trends to localized, monthly commodity fluctuations.

**Tech Stack & Tools:**
* **Visualization:** Tableau (Desktop/Public)
